# Titanic Survival Prediction - MLOps Demo

## Project Description
Predict survival of Titanic passengers using logistic regression.

## Dataset
Kaggle Titanic Dataset: https://www.kaggle.com/c/titanic/data

## Team Members
- Member 1: Data & preprocessing
- Member 2: Model training
- Member 3: API
- Member 4: Docker + CI/CD

## Project Structure
- `src/` : preprocessing, training, prediction scripts
- `data/raw` : raw CSV
- `data/processed` : preprocessed CSV
- `mlruns/` : MLflow experiments
- `tests/` : unit tests

## How to Run
```bash
python src/data_preprocessing.py
python src/train.py
```
